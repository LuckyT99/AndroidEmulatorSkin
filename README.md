# Android Emulator Skin For Honor V10
Honor V10的Android模拟器皮肤

### 使用方法
1. 下载本仓库
2. 在AVD Manager中选择模拟器
3. 点击"Edit this AVD"图标
4. 点击"Show Advanced Settings"
5. 在"Device Frame" -> "Custom skin definition"中选择本仓库文件夹
